## Hakkımda

Mekatronik mühendisliği geçmişinden gelen, uçtan uca yapay zeka uygulamaları, RAG platformları, LLMOps iş akışları ve agent tabanlı sistemler geliştiren bir Yapay Zeka Mühendisiyim. Aynı zamanda Isparta Uygulamalı Bilimler Üniversitesi'nde Mekatronik Mühendisliği yüksek lisansı yapıyorum.

Çalışmalarım; enerji sistemleri, inverter testleri, BMS teşhisi ve donanım loglarından gelen gerçek dünya mühendislik deneyimini semantik arama, veri işleme, araç kullanan agent mimarileri ve üretime hazır gözlemlenebilirlik yaklaşımıyla birleştiriyor.
